Bonança :brazil: (pronounced boh-nahn-sah) is a command-line interface (CLI) tool designed to help users manage index funds on various blockchain platforms. It provides functionalities for creating wallets, managing index funds, executing trades, and more.

Bonança is fully free and the crypto is your own. You are in full control of your funds at all times.

## Features

- Create and manage wallets using secure key vaults.
- Define and manage index funds with customizable asset allocations.
- Connect to multiple blockchains (e.g., Solana, EVM-compatible chains).
- Integrate with various exchanges and oracles for trading and price data.
- Execute trades to rebalance index funds according to target allocations.

## Quickstart

Prerequisites:

- Rust (stable) + cargo

Build the workspace:

```bash
cargo build --workspace --release
```

Run tests:

```bash
cargo test --workspace
```
